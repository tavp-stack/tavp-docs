# tavp-marketplace

Module marketplace for TAVP.

## Features

- Publish modules
- Install modules
- Version management
- Revenue tracking

## Usage

### Publish Module

```bash
tavp module:publish
```

### Install Module

```bash
tavp module:install <module-name>
```

### Manage Modules

```bash
tavp module:enable <module-name>
tavp module:disable <module-name>
tavp module:uninstall <module-name>
```
