# tavp-installer

Phalcon installer for TAVP.

## Usage

```bash
# Auto-detect PHP version
tavp phalcon:install

# Pin PHP version
tavp phalcon:install 8.3

# Pin PHP + Phalcon version
tavp phalcon:install 8.3 5.16.0
```
