# tavpkit

Starter kits for TAVP.

## Features

- Team management
- API tokens
- Profile management
- Session management
- 2FA authentication

## Install

```bash
composer require tavp/kit
tavp kit:install
```

## Upgrade

```bash
# Upgrade to App tier
tavp upgrade --to=app

# Upgrade to Enterprise tier
tavp upgrade --to=enterprise
```
