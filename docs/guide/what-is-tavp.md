# What is TAVP?

**T**ailwind CSS + **A**lpine.js + **V**olt + **P**halcon = **TAVP**

TAVP is a curated tech stack for building fast, modern PHP web applications.

## Why TAVP?

- **Performance** — 12,000+ requests/sec
- **Developer Experience** — Laravel-style CLI
- **Secure** — OTP, JWT, OAuth built-in
- **Scalable** — PHP-FPM, Swoole, RoadRunner
