# tavpkit

Starter kits untuk TAVP.

## Fitur

- Team management
- API tokens
- Profile management
- Session management
- 2FA authentication

## Install

```bash
composer require tavp/kit
tavp kit:install
```

## Upgrade

```bash
# Upgrade ke App tier
tavp upgrade --to=app

# Upgrade ke Enterprise tier
tavp upgrade --to=enterprise
```
