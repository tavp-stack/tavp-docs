# tavp-starter

Project template untuk TAVP.

## Usage

```bash
# Create project from template
composer create-project tavp/starter my-app

# Or use TAVP CLI
tavp new my-app --template=starter
```

## What's Included

- Pre-configured authentication
- Admin panel (TAVPhub)
- 40+ UI components (TAVPblocks)
- Database migrations
- Example models & controllers
- Docker setup
